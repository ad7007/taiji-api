# 6宫-9宫协作优化协议

## 问题诊断

| 宫位 | 五行 | 职责 | 负载 |
|------|------|------|------|
| 6宫 | 火 | 系统监控、告警、备份 | 0.5（低）|
| 9宫 | 火 | 行业研究、竞品分析 | 1.0（高）|

**不平衡原因**：
- 两宫都属火，缺少相生关系
- 缺少明确协作场景
- 6宫监控数据未流向9宫
- 9宫市场数据未关联6宫

---

## 协作场景设计

### 场景1：市场监控联动

```
9宫发现市场机会 → 6宫配置监控重点
     ↓
9宫竞品动态 → 6宫告警规则更新
```

### 场景2：数据双向流动

```
6宫系统数据 → 9宫分析用户行为
     ↓
9宫市场数据 → 6宫资源容量规划
```

---

## 具体行动

### 6宫增加能力

| 新增职责 | 说明 |
|----------|------|
| 市场监控 | 监控行业动态、竞品动态 |
| 数据分析支持 | 为9宫提供数据支撑 |
| 趋势预警 | 基于数据预测趋势 |

### 9宫增加能力

| 新增职责 | 说明 |
|----------|------|
| 反向监控 | 监控市场对产品的影响 |
| 容量建议 | 基于市场趋势建议资源 |
| 竞品监控 | 自动化竞品追踪 |

---

## 协作流程

```
1. 9宫采集市场数据 → 分析趋势
2. 9宫发现机会/风险 → 通知6宫
3. 6宫调整监控重点 → 加强相关指标
4. 6宫发现异常 → 通知9宫
5. 9宫分析异常原因 → 反馈5宫
```
