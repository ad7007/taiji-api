## 🚀 完整工作流

```
用户请求 → 5-中宫调度 → 并行数据收集 → 2-产品汇总 → 6-监控质检 → 5-中宫交付
```

### 执行流程

1. **5-中宫接收任务**
   - 分析任务类型
   - 分配宫位资源
   - 设置并发参数

2. **并行数据收集**（1/3/4/7/8/9 宫）
   - 1-采集：热点搜索、视频提取
   - 3-技术：模型路由、代码生成
   - 4-品牌：竞品分析、内容策略
   - 7-法务：合规检查、安全审计
   - 8-营销：用户需求、市场分析
   - 9-生态：行业调研、趋势分析

3. **2-产品质量汇总**
   - Kimi 生成报告内容
   - 内容质量优化
   - 质量评分

4. **6-监控质量检查**
   - PDF 质量评分
   - 视觉质量评分
   - 安全检查

5. **5-中宫交付**
   - PDF 生成
   - 交付客户
