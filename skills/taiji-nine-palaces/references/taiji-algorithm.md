- 基础定义

-- 阴阳分类

```
阳卦: 乾(6) 震(3) 坎(1) 艮(8)
阴卦: 坤(2) 巽(4) 离(9) 兑(7)
中宫: 5 (阴阳同体，可与任何宫组队)
```

-- 后天八卦排序

```
┌─────────┬─────────┬─────────┐
│ 4 巽(木) │ 9 离(火) │ 2 坤(土) │
├─────────┼─────────┼─────────┤
│ 3 震(木) │ 5 中(土) │ 7 兑(金) │
├─────────┼─────────┼─────────┤
│ 8 艮(土) │ 1 坎(水) │ 6 乾(金) │
└─────────┴─────────┴─────────┘
```

***

- N宫组合算法

-- 2宫：正反关系

```
阴阳配对: (阳卦, 阴卦)
例: (8,9), (3,2), (1,4)...
```

-- 3宫：三角循环

```
159: 中轴（坎-中-离）
258: 横向支撑
357: 技术闭环
456: 监控闭环
```

-- 5宫：圆形循环

```
中宫 + 4宫 = 小圆
```

-- 8宫：全模态

```
+中宫 = 全开
-中宫 = 4组正反关系
```
