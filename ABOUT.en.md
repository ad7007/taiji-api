# About Taiji API

## 🌟 Project Vision

**Taiji (太极九宫) Task Management System** is a revolutionary open-source governance framework that maps three real-world institutions into an Eastern philosophy-inspired architecture, creating a sustainable, evolvable, and verifiable operating system for open-source projects.

---

## 🏛️ Philosophy

### Eastern Wisdom Meets Modern Technology

The Taiji system draws inspiration from ancient Chinese philosophy:

- **Taiji (太极)**: The supreme ultimate that generates all things
- **Nine Palaces (九宫)**: A 3x3 grid representing different aspects of governance
- **Yin-Yang (阴阳)**: Balance between opposing forces
- **Five Elements (五行)**: Interconnected cycles of generation and restriction

### Three Real-World Institutions

The system maps three fundamental institutions:

1. **Government** - Decision-making and execution (Palace 5 - Central Control)
2. **Market** - Resource allocation and competition (Palaces 1-4, 6-9)
3. **Community** - Collaboration and co-governance (All palaces working together)

---

## 🎯 Core Features

### 1. Nine Palaces Architecture

Each palace has specific responsibilities:

| Palace | Name | Responsibilities | Element |
|--------|------|------------------|---------|
| 1 | Data Collection | Web scraping, data gathering | Earth |
| 2 | Product Quality | Document management, quality control | Metal |
| 3 | Tech Team | Model allocation, code management | Wood |
| 4 | Brand Strategy | Identity management, external image | Water |
| 5 | Central Control | Command coordination, task scheduling | Earth |
| 6 | IoT Monitoring | System monitoring, config backup | Fire |
| 7 | Legal Framework | Security scanning, TDD acceptance | Metal |
| 8 | Marketing & Service | Customer service, marketing | Wood |
| 9 | Ecosystem | Ecosystem building, partnerships | Earth |

### 2. L4 Rule Engine

Advanced governance layer featuring:

- **Palace 5 Commander**: Perception → Decision → Allocation → Closed-loop
- **Palace 7 TDD**: Red light → Execution → Green light → Delivery

### 3. AI Integration

- **Zero Token Mode**: Zero-cost access to mainstream AI models
- **Model Allocation**: Intelligent model selection based on task type
- **Cost Optimization**: Automatic cost-saving recommendations

### 4. Community Governance

- **Co-Maintenance Model**: Community-owned and operated
- **Founder Retention**: Founder retains supreme authority for direction
- **Transparent Decision-making**: All decisions recorded and public
- **Meritocratic Promotion**: Contribution determines influence

## 🚀 Technical Stack

### Backend

- **Framework**: FastAPI (Python 3.8+)
- **Database**: SQLite / PostgreSQL (planned)
- **Cache**: Redis (optional)
- **Task Queue**: Celery (planned)

### AI & Automation

- **OpenClaw**: AI agent framework
- **Crawlee**: Web scraping automation
- **Multiple Models**: Claude, DeepSeek, GPT, Gemini, Qwen

### DevOps

- **Container**: Docker & Docker Compose
- **CI/CD**: GitHub Actions
- **Monitoring**: Built-in health checks
- **Backup**: Rclone cloud backup
