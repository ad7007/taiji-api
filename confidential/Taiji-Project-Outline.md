【基础定义】
- 中宫(5): 阴阳同体,可与任何宫组队
- 阳卦: 乾(6)震(3)坎(1)艮(8)
- 阴卦: 坤(2)巽(4)离(9)兑(7)

【线程阴阳两端特性 - 深度公式】

每个线程 T 具有阴阳双端:
  T = (T_yin, T_yang) 其中 T_yin ∈ [0,1], T_yang ∈ [0,1]
  
线程阴阳度量:
  YinYang_Balance(T) = T_yang - T_yin ∈ [-1, +1]
  
  当 YinYang_Balance(T) > 0: 线程偏阳(发散)
  当 YinYang_Balance(T) < 0: 线程偏阴(收敛)
  当 YinYang_Balance(T) = 0: 阴阳平衡(中宫态)

线程转换函数:
  Transform(T, direction) = {
    if direction == 'YIN→YANG':
      T_yin *= (1 - α)    // 阴端衰减
      T_yang += T_yin × α // 阴转阳
    if direction == 'YANG→YIN':
      T_yang *= (1 - α)   // 阳端衰减
      T_yin += T_yang × α // 阳转阴
  }
  其中 α ∈ (0,1) 为转换系数

【24线程阴阳拓扑结构】

线程连接图 G = (V, E) 其中:
  V = {1,2,3,4,6,7,8,9}  // 8个宫位节点
  E = {(u,v) | u∈阴卦, v∈阳卦}  // 阴阳连接边

阴24线程端点: V_yin = {1☵, 2☷, 6☰, 8☶}   // 水土金土

线程连接矩阵 M[8×8]:
  M[i][j] = {
    1  if (i,j) ∈ E 且 相生(i,j)
    -1 if (i,j) ∈ E 且 相克(i,j)
    0  otherwise
  }

【N宫组合数学公式】

定义:
  P = {1,2,3,4,5,6,7,8,9}  // 宫位全集
  P_yang = {3,4,7,9}        // 阳卦集
  P_yin = {1,2,6,8}         // 阴卦集
  P_center = {5}            // 中宫

组合函数 C(n) = {S ⊆ P | |S| = n}  // n宫的所有组合

1宫组合 (中宫):
  C(1) = {{5}}
  特性: 阴阳同体, ∀x∈P, Pair(5,x) = VALID

2宫组合 (正反关系):
  C(2) = {{p_i, p_j} | p_i ∈ P_yin, p_j ∈ P_yang}
  |C(2)| = C(4,1) × C(4,1) = 16 组
  
  关系判定:
    Relation(p_i, p_j) = {
      '相生' if (p_i → p_j) ∈ 相生链
      '相克' if (p_i → p_j) ∈ 相克链
      '中性' otherwise
    }

3宫组合 (三角循环):
  C(3) = {159, 258, 357, 456}  // 预定义稳定三角
  
  三角稳定性公式:
    Stability(Δ) = Σ(i,j)∈Δ M[i][j] / 3
    
    当 Stability(Δ) > 0: 相生三角 (稳定)
    当 Stability(Δ) < 0: 相克三角 (不稳定)
    
  扩展组合 (所有3宫排列):
    C(3)_all = C(9,3) = 84 组
    有效三角 = {t ∈ C(3)_all | Stability(t) ≠ 0}

4宫组合 (2×2正反):
  C(4) = {S | S = {p1,p2,p3,p4}, ∃划分 (A,B) 使 |A|=|B|=2, A⊆P_yin, B⊆P_yang}
  
  实质: 2组正反关系
  Structure(4) = Pair(p_i, p_j) × Pair(p_k, p_l)
  其中 (p_i,p_k) ∈ P_yin, (p_j,p_l) ∈ P_yang

5宫组合 (小圆循环):
  C(5) = {S ∪ {5} | S ∈ C(4)}
  
  圆形循环公式:
    Circle(S, 5) = {
      path: p1 → 5 → p2 → p3 → 5 → p4 → p1
      length: 6步
      center: 5 (中宫作为跳点)
    }

6宫组合 (中圆):
  C(6) = {S | S ⊂ P, |S| = 6, 5 ∉ S} ∪ {S ∪ {5} | |S| = 5}
  
  结构分解:
    Structure(6) = {
      '2宫对': 1组,
      '3宫三角': 2组,

    }

7宫组合 (大圆):
  C(7) = {S | S ⊂ P, |S| = 7}
  
  Structure(7) = Structure(6) + 1个浮动宫位
  浮动宫位可插入任意位置形成新循环

8宫组合 (全模态/纯八卦):
  C(8) = {P - {5}}  // 不含中宫的8宫
  
  全模态 (8+5):

【相生循环算法】

相生函数 S(x):
  S(3) = 6   // 木→金 (震生乾)
  S(6) = 9   // 金→火 (乾生离)
  S(9) = 7   // 火→金 (离生兑)
  S(7) = 4   // 金→木 (兑生巽)
  S(4) = 8   // 木→土 (巽生艮)
  S(8) = 3   // 土→木 (艮生震)
  S(5) = x   // 中宫生任何宫
  
相生链闭合性:
  S⁶(3) = 3  // 6步相生回到起点
  
路径计算:
  Path_sheng(x, y) = min{n | Sⁿ(x) = y}
  Distance_sheng(x, y) = Path_sheng(x, y) mod 6

【相克循环算法】

相克矩阵 K[9×9]:
  K[i][j] = 1 if (i,j) ∈ {(3,6),(6,9),(9,7),(7,4),(4,8),(8,1),(1,3)}
  
相克强度:
  Strength(i,j) = {
    2  if 五行(i) 克 五行(j) 且 阴阳(i) = 阴阳(j)  // 同阴阳相克更强
    1  if 五行(i) 克 五行(j) 且 阴阳(i) ≠ 阴阳(j)  // 异阴阳相克较弱
    0  otherwise
  }
