## 🔧 核心循环算法公式

```
【基础定义】
- 中宫(5): 阴阳同体,可与任何宫组队
- 阳卦: 乾(6)震(3)坎(1)艮(8)
- 阴卦: 坤(2)巽(4)离(9)兑(7)

【线程阴阳两端特性 - 深度公式】

每个线程 T 具有阴阳双端:
  T = (T_yin, T_yang) 其中 T_yin ∈ [0,1], T_yang ∈ [0,1]
  
线程阴阳度量:
  YinYang_Balance(T) = T_yang - T_yin ∈ [-1, +1]
  
  当 YinYang_Balance(T) > 0: 线程偏阳(发散)
  当 YinYang_Balance(T) < 0: 线程偏阴(收敛)
  当 YinYang_Balance(T) = 0: 阴阳平衡(中宫态)

【24线程阴阳拓扑结构】

线程连接图 G = (V, E) 其中:
  V = {1,2,3,4,6,7,8,9}  // 8个宫位节点
  E = {(u,v) | u∈阴卦, v∈阳卦}  // 阴阳连接边

阳24线程端点: V_yang = {3☳, 4☴, 7☱, 9☲}  // 木木金火
阴24线程端点: V_yin = {1☵, 2☷, 6☰, 8☶}   // 水土金土
```
