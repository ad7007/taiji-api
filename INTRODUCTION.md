## 🌟 项目愿景

**太极 (太极九宫) 任务管理系统**是一个革命性的开源治理框架，将现实世界的三种制度映射到东方哲学启发的架构中，为开源项目创建一个可持续、可进化、可验证的操作系统。

---

## 📖 项目背景

### 为什么创建太极 API？

当今开源项目面临三大挑战：

1. **维护者负担过重** - 少数人承担大部分工作
2. **社区参与度低** - 贡献者难以持续参与
3. **项目治理混乱** - 决策不透明，权力过于集中

太极 API 通过**九宫格治理架构**解决这些问题：

- 🎯 **九宫分工** - 每个宫位有明确职责
- ⚖️ **阴阳平衡** - 自动检测和调节负载
- 🤝 **社区共治** - 创始人保留最终权限，日常运营由社区共同维护

---

## 🏛️ 九宫格架构

太极系统将任务管理分为 9 个宫位，每个宫位对应不同的职责：

```
┌─────────────┬─────────────┬─────────────┐
│ 4-品牌战略   │ 9-行业生态   │ 2-产品质量   │
│ 水 · 巽宫     │ 土 · 离宫     │ 金 · 坤宫     │
├─────────────┼─────────────┼─────────────┤
│ 3-技术团队   │ 5-中央控制   │ 7-法务框架   │
│ 木 · 震宫     │ 土 · 中宫     │ 金 · 兑宫     │
├─────────────┼─────────────┼─────────────┤
│ 8-营销客服   │ 1-数据采集   │ 6-物联监控   │
│ 木 · 艮宫     │ 土 · 坎宫     │ 火 · 乾宫     │
└─────────────┴─────────────┴─────────────┘
```

### 各宫职责

| 宫位 | 名称 | 职责 | 五行 |
|------|------|------|------|
| **1 宫** | 数据采集 | 网页抓取、数据收集、信息整合 | 土 |
| **2 宫** | 产品质量 | 文档管理、质量把控、飞书集成 | 金 |
| **3 宫** | 技术团队 | 模型分配、代码管理、GitHub 集成 | 木 |
| **4 宫** | 品牌战略 | 身份管理、对外形象、虚拟身份 | 水 |
| **5 宫** | 中央控制 | 指挥协调、任务调度、L4 规则引擎 | 土 |
| **6 宫** | 物联监控 | 系统监控、配置备份、定时任务 | 火 |
| **7 宫** | 法务框架 | 安全扫描、TDD 验收、代码审计 | 金 |
| **8 宫** | 营销客服 | 客户服务、营销自动化、工单系统 | 木 |
| **9 宫** | 行业生态 | 生态建设、合作伙伴、插件市场 | 土 |

## ✨ 核心功能

### 1. L4 规则引擎

**5 宫指挥官 + 7 宫 TDD 验收**的完整闭环：

```
余总指令
    ↓
5 宫感知 → 优先级判断 → 宫位分配
    ↓
7 宫 TDD → 定义验收标准
    ↓
1-9 宫执行 → 完成任务
    ↓
7 宫验收 → 绿灯检查
    ↓
5 宫交付
```

**特点**:
- 🔴 **红灯确认** - 任务开始前确认失败状态
- 🟢 **绿灯交付** - 完成后对照标准验收
- 🔄 **返工机制** - 未通过自动返工

### 2. Zero Token 零成本 AI

**3 宫模型分配**支持：

| 模式 | 说明 | 成本 |
|------|------|------|
| **API Token** | 传统 API 调用 | 付费 |
| **Zero Token** | 浏览器自动化 | 免费 ✅ |

**支持模型**:
- Claude (Sonnet/Opus)
- DeepSeek
- GPT-4o
- Gemini
- Qwen (通义千问)

**成本优化**: 日常任务优先使用 Zero Token 模式，可节省 100% API 费用！

### 3. Crawlee 智能数据抓取

**1 宫数据采集**增强：

- 🕷️ **双模式切换** - HTTP 快速 + 浏览器渲染
- 🛡️ **防封禁特性** - IP 轮换、鼠标模拟、UA 轮换
- 📊 **结构化输出** - 自动整理成 Excel/JSON

**比喻**: "就像 100 个穿不同衣服的人轮流进去拿货，谁也不会引起注意"

### 4. 定时备份系统

**6 宫物联监控**负责：

- ⏰ **定时备份** - 中午 12 点 + 晚上 23 点
- ☁️ **云端存储** - Rclone 集成（Google Drive/OneDrive 等）
- 🔒 **加密备份** - 支持加密远程存储

### 5. 社区共同治理

**创新治理模式**:

- 👑 **创始人保留最高权限** - 方向把控、最终决策
- 🤝 **社区共同维护** - 日常运营由社区负责
- 🗳️ **民主决策** - 重大决定社区投票
- 📋 **透明公开** - 所有决策记录公开

## 🚀 快速开始

### 安装

```bash
# 克隆项目（国内用户推荐 Gitee）
git clone https://gitee.com/miroeta/taiji-api.git
cd taiji-api

# 或 GitHub
git clone https://github.com/ad7007/taiji-api.git
cd taiji-api

# 安装依赖
pip install -r requirements.txt
```

### 启动服务

```bash
# 启动 API
python -m uvicorn api.taiji_api:app --host 0.0.0.0 --port 8000

# 开发模式（自动重载）
python -m uvicorn api.taiji_api:app --reload
```

### 访问文档

```
Swagger UI: http://localhost:8000/docs
ReDoc:      http://localhost:8000/redoc
```

### 示例：创建任务

```bash
# 调用 5 宫指挥官
curl -X POST http://localhost:8000/api/l4/command \
  -H "Content-Type: application/json" \
  -d '{"command": "分析这份销售数据"}'
```
