# 🚀 5 分钟快速开始贡献

> **无需复杂流程，5 分钟就能开始贡献！**

---

## ⏱️ 时间线

```
0:00 - 打开 GitHub
1:00 - Fork 项目
2:00 - 克隆到本地
3:00 - 创建分支
4:00 - 修改代码
5:00 - 提交 PR ✅
```

---

## 📝 步骤详解

### 第 1 分钟：Fork 项目

1. 访问 https://github.com/ad7007/taiji-api
2. 点击右上角 **Fork** 按钮
3. 等待 Fork 完成

**完成!** ✅ 你现在有自己的副本了

---

### 第 2 分钟：克隆到本地

```bash
# 替换为你的用户名
git clone https://github.com/YOUR_USERNAME/taiji-api.git
cd taiji-api
```

**完成!** ✅ 代码已在本地

---

### 第 3 分钟：创建分支

```bash
# 创建新分支
git checkout -b fix/typo

# 或者
git checkout -b feature/new-feature
```

**完成!** ✅ 准备工作就绪

---

### 第 4 分钟：修改代码

**简单贡献示例**（选一个）：

#### 修复文档错别字

```bash
# 用编辑器打开 README.md
nano README.md

# 修改错别字，保存
```

#### 添加注释

```python
# 打开任意 Python 文件
# 添加函数注释
def calculate_balance(palace_loads):
    """
    计算阴阳平衡度
    
    Args:
        palace_loads: 宫位负载字典
    
    Returns:
        平衡度 (0-1)
    """
    pass
```

#### 修复简单 Bug

找到 Issue 中标记为 `good first issue` 的问题，修复它。

**完成!** ✅ 代码已修改

---

### 第 5 分钟：提交 PR

```bash
# 添加修改
git add .

# 提交
git commit -m "docs: 修复 README 错别字"

# 推送
git push origin fix/typo
```

然后在 GitHub 上：

1. 访问你的 Fork 页面
2. 点击 **Compare & pull request**
3. 填写 PR 描述
4. 点击 **Create pull request**

**完成!** ✅ 第一个 PR 已提交

---

## 🎉 恭喜！

你已经完成了第一个贡献！

**接下来**:
- 等待 Review（通常 1-3 天）
- 根据反馈修改（如需要）
- PR 合并后，名字列入贡献者名单
