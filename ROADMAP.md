# 太极 API 路线图

本文档记录太极 API 的发展规划和版本演进。

---

## 🎯 v2.0 (当前版本) - 2026-03-18 ✅

### 核心功能

- ✅ L4 规则层集成
  - 5 宫指挥官（感知→决策→调配→闭环）
  - 7 宫 TDD 验收（红灯→执行→绿灯→交付）
- ✅ 3 宫模型分配
  - API Token vs Zero Token 双模式
  - 支持 Claude/DeepSeek/GPT/Gemini/Qwen
- ✅ 1 宫数据采集
  - Crawlee 集成
  - HTTP 快速模式 + 浏览器渲染模式
  - 防封禁特性（IP 轮换、鼠标模拟）
- ✅ 定时备份系统
  - 中午 12 点 + 晚上 23 点自动备份
  - Rclone 云端存储

### 九宫格实现

- ✅ 1 宫（数据采集）- Crawlee 增强
- ✅ 2 宫（产品质量）- 飞书文档集成
- ✅ 3 宫（技术团队）- 模型分配
- ✅ 4 宫（品牌战略）- 身份管理
- ✅ 5 宫（中央控制）- 指挥官
- ✅ 6 宫（物联监控）- 备份配置
- ✅ 7 宫（法务框架）- TDD 验收
- ⏳ 8 宫（营销客服）- 待实现
- ⏳ 9 宫（行业生态）- 待实现

### 开源建设

- ✅ MIT 许可证
- ✅ README.md
- ✅ CONTRIBUTING.md
- ✅ Issue/PR 模板
- ✅ GitHub 仓库创建
