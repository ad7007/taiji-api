# 太极API推荐奖励机制

## 推荐规则

### 一级推荐
- 推荐1人付费 → 获得10%返佣
- 推荐3人付费 → 获得15%返佣
- 推荐5人付费 → 获得20%返佣

### 二级推荐
- 一级推荐的客户再推荐 → 获得5%返佣

## 奖励发放

- 每月结算一次
- 自动打入账户余额
- 可提现或抵扣续费
