# 社群自动互动方案

## 一、触发条件
• 有人提到"太极/OpenClaw/智能体"
• 有人提问AI相关问题
• 有人分享AI项目

## 二、响应模板
1. 认可式："这个思路不错！太极系统也有类似的设计..."
2. 补充式："补充一点，我们测试发现..."
3. 提问式："有个问题想请教，你们是怎么处理...？"
4. 分享式："刚好我们在做这个，分享下我们的方案..."

## 三、频率控制
• 同一群每天最多3次
• 间隔至少2小时
• 避免刷屏感

## 四、人工介入
• 复杂问题标记，人工跟进
• 负面反馈及时上报
• 重要线索转余总

• 微信群配置
• QQ群配置
• 其他社群配置
